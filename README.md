# 共享汽车管理系统简介 java471

## 项目概述

共享汽车管理系统是基于Springboot+Vue的前后端分离项目。它采用了MVC三层体系结构，结合Spring、SpringBoot、Mybatis和Vue等技术开发而成。本系统充分利用了B/S结构，通过Web端提供了便捷的用户操作界面。

## 技术栈

- 前端：Vue.js
- 后端：Springboot
- 数据库：MySQL
- 项目构建：Maven

## 功能模块

- 用户管理
- 车辆管理
- 订单管理
- 费用结算
- 数据统计

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：支持现代浏览器的最新版本
- 后端：JDK 1.8 或更高版本，Springboot 环境搭建

## 部署步骤

1. 准备运行环境，包括Java开发环境和MySQL数据库。
2. 使用Maven构建项目。
3. 部署后端服务。
4. 部署前端服务。
